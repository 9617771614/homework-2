# homework-2

Код выполнения домашнего задания №2

Код написан и проверен с помощью https://godbolt.org/

В данном коде реализована:
•	функция вывода развернутой строки, которую программа получила через консоль. 
•	не закончена реализация функции калькулятора (реализован функционал ввода и контроля вводимых символов).

А. Для вывода введенной строки от последнего элемента к первому необходимо.
Для этого необходимо ввести в консоль:
1) «1» - для выбора режима разворота строки (2 для режима калькулятор, другие символы игнорируются);
2) «Y» - для подтверждения перехода к вводу строки (если ввести другой символ, программа вернется к выбору режима работы)
3) ввести любые символы кроме «/», количество символов не более 20 (20 – сделано для проверки работы ограничения количества символов)
4) по окончанию ввода ввести символ окончания ввода - «/»

Б. Для реализации функции калькулятора сделано следующее:
1) при вводе «2» программа переходит к калькулятору;
2) реализовано подтверждение входа в режим калькулятора (требуется ввести «Y», другие символы вернут к выбору режима работы.
3) реализован ввод числа А и Б (число А и Б не более 5 символов т. е. макс 99999) 
4) реализован контроль вводимых символов т.е. если ввести не цифру а букву или символ, то программа выдаст ошибку.
